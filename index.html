Mengen Info Downloader

このアプリは、YouTubeの動画URLのリストを入力すると、その動画のタイトル、公開日時(UTC+0)、説明文をテキストファイルとしてまとめたZIPファイルを生成してダウンロードするツールです。

私はこのアプリケーションに一切の悪意を含んでいないことを保証しますが、

WindowsなどのOSの自動保護機能により、解凍時に警告が表示されることがあります。

Windowsの場合は解凍前にzipのプロパティを変更するか、それでも解決しない場合あるいは他のOSの場合は7-Zipなどの解凍ソフトを使用して解凍してください。

URLの形式

基本的にはYouTubeから共有したURLをそのまま貼れば問題ありませんが、もし動かなかった場合は、以下の形式を推奨します。

https://www.youtube.com/watch?v={VideoID}
例: https://www.youtube.com/watch?v=LF87NAkWkZo
VideoIDの形式:
ここで、{VideoID}はYouTube上で一意に識別される動画のIDで、以下の形式です。
英数字とハイフン、アンダースコアを組み合わせる可能性のある11文字で構成(すべて半角)
末尾の?si=xxxxはないことを確認してください。

APIについて

このアプリはYouTube Data API v3を使用して動画の情報を取得しています。

キーの発行についてはYouTube Data API v3の公式ドキュメントをご確認ください。

このアプリはユーザー認証を必要としません。

APIにはクオータ制限があります。詳細はYouTube Data API v3の公式ドキュメントをご確認ください。

このアプリでは、クオータを無駄にしないために、1回のリクエストで最大50個の動画情報を取得しており、

それを1回または複数回実行しています。
